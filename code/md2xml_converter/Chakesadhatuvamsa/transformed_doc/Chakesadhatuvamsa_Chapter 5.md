**अद्धिकनाविकथूपवंसो**

# 1. ततो उत्तराय दिसाय सब्बे अरहन्ता गन्त्वा साखापल्लवलङ्कते सितच्छाये एकस्मिं निग्रोधरुक्खे अधिवत्था देवता ते आगच्छन्ते दिस्वा किं भन्ते इध आगच्छथाति पुच्छित्वा ततो तेहि आवुसो इमं ठानं सप्पायं, इमस्मिं ठाने एकं जिनवरकेसधातुं ठपनत्थाय आगतम्हाति वुत्ते साधु भन्ते अहम्पि तुम्हेहि सद्धिं अनुमोदनं करेय्यन्ति आह।
# 2. अथ खो आयस्मा गुणसागरो धातुपट्ठाकपरियेसने ममेव भारोति धातुपट्ठाकत्थाय अञ्जलिं पग्गहेत्वा अधिट्ठानं करोन्तो इमं गाथमाह -

“दायको सुलभो होतु, सचे त्वं मुनिमुद्धज।

हिताय सब्बसत्तस्स, ठस्ससि सततं इध।”

एवं वत्वा तं धातुं नमस्समानोव निसीदि।

# 3. तस्मिं खणे जिनधातुया आनुभावेन थेराधिट्ठानबलेन च सत्थुआदीनं आनुभावेन च मन्दवातेन आनीता विञ्ञातसासनं वाणिजपरिपुण्णा एका नावा तस्मिंयेव पदेसे अनुपत्ता अहोसि।
# 4. अथ खो वाणिजा भिक्खुसङ्घं दिस्वा “मयं समुद्दचरा दुल्लभं छन्नं खीणासवानं सन्तिकं उपगन्त्वा सङ्घस्स दस्सन”न्ति चिन्तेत्वा नाविकं आरोचेसुं।
# 5. नाविकोपि तं सुत्वा सङ्घे पसन्नमानसो सहवाणिजेहि महानावाय ओरुय्ह खुद्दकनावं अभिरूहित्वा तेसं छन्नं खीणासवानं सन्तिकं उपगन्त्वा सङ्घं वन्दित्वा “किं इध अय्यानं किच्चं अत्थी”ति पुच्छि।
# 6. “महन्तं खो उपासक एकं सत्थुसरीरजं धातुं लोकहितत्थाय ठपनत्थं इधगता”ति।
# 7. तं सुत्वा नाविको तुट्ठमानसो ते वाणिजे आमन्तेत्वा “लाभा वत भो सुलद्धं वत भो अम्हेहि ये मयं एवं महिद्धिकं जनहितावहं सत्थुसरीरधातुं पस्सामा”ति। “महासमुद्दस्मिं भो चिरकालं सञ्चरन्ता ईदिसं लाभं न लद्धपुब्ब”न्ति। “तुम्हे मम सहाया भवथ अहमेव थूपं करेय्य”न्ति आह।
# 8. ते पि खो वाणिजा साधूति सम्पटिच्छिंसु, रुक्खदेवतापि मनुस्सवेसं गहेत्वा ‘अहम्पि तुम्हाकं सहायो भवेय्य’न्ति आह।
# 9. ते सब्बेपि थेरानं सन्तिकं उपसङ्कमित्वा यथानुरूपं दानं दत्वा थेरे वन्दित्वा निसीदिंसु।
# 10. अथ खो नाविको “अहमेव भन्ते धातुथूपकरणे उस्सुक्कं करेय्य”न्ति याचि।
# 11. थेरापि “साधु उपासक करोही”ति अहंसु।
# 12. नाविको पुरिसे आणापेत्वा बहुधनम्पि विस्सज्जेत्वा धातुगब्भं करित्वा तस्मिं गब्भे सत्तरतनानि पूरेत्वा गब्भस्स मज्झे बुद्धारहं आसनं पञ्ञापेत्वा गुणसागरथेरस्स हत्थतो तं धातुं गहेत्वा ठपेसि।
# 13. सहठपनेनेव सब्बानि अच्छरियानि पातुरहेसुं।
# 14. अथ खो नाविको पहट्ठचित्तो इमं गाथमाह -

“लोकाचरियभूतस्स असमस्स महेसिनो।

धातुया दायको अस्मि लाभा मे वत मारिसाति॥”

# 15. एवं वत्वा चेतियं पञ्चपतिट्ठितेन वन्दित्वा नानाविधधजपटाकं उस्सापेत्वा चेतियं पदक्खिणं कत्वा अरहन्तानं पादे वन्दित्वा वाणिजेहि परिवुतो नावं आरुहित्वा अत्तनो नगरं निय्यासि।
# 16. रुक्खदेवता चेतियं तुसित्वा यावज्जकालापि दीघायुका हुत्वा तं चेतियं रक्खति।

**अद्धिकनाविकथूपवंसो निट्ठितो।**
