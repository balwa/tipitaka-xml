**सत्तनाविकथूपवंसो**

# 1. ततो उत्तराय दिसाय गन्त्वा खीणासवा भिक्खू नच्चासन्नं नातिदूरं अत्थिकानं गमनागमनसम्पन्नं एकं ठानं दिस्वा इदं हि खो हि खो आवुसो ठानं सप्पायं, इमस्मिं ठाने अवसिट्ठं एकं जिनकेसधातुं ठपेतब्बन्ति।
# 2. अथ खो आयस्मतो रेवतस्स एतदहोसि – “सब्बेसं पञ्चन्नं अरहन्तानं मनोरथो मत्थकं पत्तो धातुदायकपटिलाभेन। तथाहम्पि इमस्मिं पदेसे एकं दायकं लभेय्यं। यंनूनाहं धातुथूपदायकत्थाय पत्थनं करेय्य”न्ति।
# 3. अथ खो रेवतथेरो सत्थरि परमउक्कं सपत्तं आदरं सम्पादेत्वा एवमाह – “सचे भन्ते कप्पसतसहस्साधिकानि चत्तारि असङ्ख्येय्यानि पारमियो पूरेत्वा पञ्चमहापरिच्चागं परिच्चजित्वा देवमनुस्सेहि दुक्करं कम्मं करित्वा बुद्धभावं पत्तोसि, अज्ज तवानुभावेन थूपदायको इधानुप्पत्तो होतू”ति अधिट्ठानं कत्वा इमं गाथमाह –

“सचे लोकहितत्थाय, अनुजानिध नायको।

धातुथूपस्स दायकं, लभेय्य तेजसा तवा”ति॥

# 4. एवं पत्थयन्तस्सेव दमिळवाणिजा सद्धासम्पन्ना सत्त महानावा जिनधातुतेजेन आनीता विय हुत्वा अञ्ञं देसं गन्तुं असमत्था तस्मिं येव ठाने अनुप्पत्ता अहेसुं।
# 5. तेपि खो वाणिजा दूरतोव भिक्खुसङ्घं दिस्वा दानं दातुकामा जेट्ठनाविकं आरोचेसुं।
# 6. तं सुत्वा नाविको तेनहि अहम्पि गमिस्सामीति वत्वा अञ्ञेहि नाविकेहि परिवुतो तेसं सन्तिकं गन्त्वा खीणासवानं दानं दत्वा पयिरुपासित्वा ते खीणासवे पुच्छिंसु “किं पनिध भन्ते आगतत्था”ति।
# 7. “दसबलस्स पुरिसाजञ्ञस्स भगवतो केसधातुं पाणीनं हिताय ठपेतुं आगतम्हा”ति अहंसु।
# 8. अथ खो तेसं सत्तन्नं नाविकानं वाणिजानञ्च एतदहोसि - “मयंहि पुब्बे भगवतो दस्सनकाले छब्बण्णरंसीहि जलमानसरीरं पस्सिम्हा। इदानि भगवतो सिरजकेसधातुया आनुभावो कीदिसो”ति चिन्तयन्तानं कङ्खा उप्पज्जति।
# 9. तेसं कङ्खच्छेदनत्थं केसधातु पाटिहारियं अकासि।
# 10. तेनवुत्तं -

“दस्सेति धातुया रस्मि, यमकं पाटिहारियं।

कण्डम्बमूले पत्ताव, लोकचक्खुविनयन”न्ति॥

# 11. अथ खो तेपि नाविका वाणिजा च तं पाटिहारियं दिस्वा सञ्जातसद्धा हुत्वा अत्तनो अत्तनो अनुरूपं पूजासक्कारं करिंसु। धातुया सक्कारं कत्वा खीणासवानं सन्तिकं गन्त्वा थूपकारणोकासं याचित्वा खीणासवेहि दिन्नोकासा ते नाविका वाणिजा वुत्तनयेनेव धातुगब्भं कारेत्वा तस्मिं सब्बरतनं रासिं कत्वा बुद्धारहं आसनं पञ्ञापेत्वा तेलपदीपं जालेत्वा बुद्धबिप्पं असितिसावकानं बिम्बञ्च कारेत्वा मज्झे सुवण्णचङ्कोटकं ठपेत्वा रेवतथेरस्स हत्थतो अनोपमं केसधातुं सम्पटिच्छित्वा गन्धोदकेन न्हापेत्वा सुवण्णरजतमयेहि पुप्फेहि दीपेहि च धातुं खमापेत्वा चङ्कोटके ठपेसुं।
# 12. तं खणञ्ञेव हेट्ठा वुत्तप्पकारानि पथवी कम्पनादीनि अच्छरियानि पातुरहेसुं। दससहस्सचक्कवाळदेवता साधुकारं अदंसु।
# 13. ते नाविका चेतियं बन्धित्वा निट्ठिते चेतिये अनेकप्पकारं धजपटाकं समुस्सापेत्वा चेतियं वन्दित्वा “भन्ते मयं दमिळउपासका अञ्ञं सम्फस्सतं केसधातुं सद्दहित्वा एवरूपं करोमा”ति आहंसु।
# 14. ततो तं चेतियं सद्धाय कारितत्ता सद्धाचेतियन्ति वदन्ति। दमिळेहि कारितत्ता दमिळचेतियन्तिपि वदन्ति।
# 15. तेपि खो नाविका वाणिजा चेतियं खमापेत्वा अरहन्तानं दानं दत्वा सकसकनावं आरुय्ह पक्कमिंसु।
# 16. छ खीणासवा परिपुण्णसङ्कप्पा अहेसुं।

**सत्तनाविकथूपवंसो निट्ठितो।**

# 17. तम्पिदेसं केसधातूनं निवासनट्ठानत्ता केसवतीति नामं आलपति।

**छकेसधातुवंसो निट्ठितो।**

केसधातुवंसो गन्थो, सब्बजनपसादको।

यं मया लिखितो तेन, पञ्ञवा होमि नागते॥

सद्धाधनं सीलधनं, हिरिओत्तप्पियं धनं।

सुतधनञ्च चागो च, पञ्ञा वे सत्तमं धनं॥

इमेहि सत्तधनेहि, सम्पन्नो च भवामहं।

इच्छितब्बं वरं मय्हं, जन्तुपुण्णोव चन्दिमा॥

रतनं देवयामानं, भोन्तो तेसं करोमहं।

सहस्सानंहि तं गेहं, इन्दब्रह्माहि पूजितं॥

(ञानन्दवंस)
