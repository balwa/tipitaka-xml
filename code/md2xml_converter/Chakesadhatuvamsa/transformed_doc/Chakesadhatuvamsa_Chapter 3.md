**पज्जुन्नथूपवंसो**

# 1. तस्स अविदूरे दक्खिणदिसाभागे तिगावुतन्तरं मग्गं गन्त्वा सब्बे छ खीणासवा नानाविधवालुकरासिपरिकिण्णं परमरमणीयं केलासपब्बतसन्निभं रुक्खतिणपासाणसक्खरकठलरहितं कतसुधाकम्ममिवङ्गणं सकलजनरमणीयं एकं ठानं दिस्वा चिन्तयिंसु – ‘सप्पायमिदं खो आवुसो ठानं, इमस्मिं एकं केसधातुं अम्हेहि ठपेतब्ब’न्ति।
# 2. अथ खो अयस्मा सोभितो “इमस्स धातुस्स दायकपरियेसने मम भारो”ति वत्वा एकंसं उत्तरासङ्गं करित्वा जाणुमण्डलं पथवियं निधाय अञ्जलिं पग्गय्ह जिनधातुवरं वन्दित्वा पत्थनं करोन्तो इमं गाथमाह -

दायको पतिपात्वज्ज, तेजसा तव नायक।

सब्बसत्तहितत्थाय, इध ठस्सति चे वरं॥”

# 3. इच्चेवं परमनिपच्चाकारं कत्वा पत्थनं करोन्तस्सेव वलाहकगब्भतो पज्जुन्नो नाम देवपुत्तो महन्तेहि परिवारेहि परिवुतो नानाविधं परमदस्सनीयतरं अलङ्कारं अलङ्करित्वा सकविमानतो ओरुय्ह तस्स पुरतो पातुरहोसि।
# 4. अथ खो आयस्मा सोभितो तं दिस्वा “उपासक त्वं उळारानुभावो, अङ्गीरसस्स अप्पटिमस्स भगवतो धातुदायको होही”ति आह।
# 5. पज्जुन्नो महाराजापि ता वचनं सुत्वा “साधु भन्ते, अहमेव तस्स दायको भवेय्य”न्ति वत्वा “तेनहि त्वं उपासक थूपस्स बन्धनट्ठानं विचारेही”ति आह।
# 6. तस्मिं खणे पज्जुन्नो महाराजा इद्धिया एकं आवाटं करित्वा तत्थ धातुगब्भं मापेत्वा तस्मिं सत्तरतनमयं आसनं पञ्ञापेत्वा वुत्तप्पकारं असीतिमहासावकानं बिम्बं बुद्धबिम्बं बुद्धमातापितूनं बिम्बञ्च सुवण्णमयं पटिसङ्खरित्वा आसनमज्झे सत्तरतनमयं धातुचङ्कोटकं ठपेत्वा तं जिनधातुवरं छब्बण्णरंसीहि विज्जोतमानं दिब्बगन्धोदकेन न्हापेत्वा ‘इधेव भन्ते तिट्ठतु सकलजनहिताया’ति वत्वा छन्नं अरहन्तानं रुचिं गहेत्वा तस्मिं येव चङ्कोटके ठपेसि।
# 7. तस्मिं खणे हेट्ठा वुत्तप्पकारानि पथवीकम्पनादीनि अच्छरियानि अहेसुं।
# 8. अथ खो पज्जुन्नो महाराजा धातुवरं सञ्जोतपेमो हुत्वा इमं गाथमाह -

“एवं महनुभावस्स, असमस्स महेसिनो।

धातुया दायको अस्मि, अहो मे पुञ्ञसञ्चयो॥”

एवं वत्वा भगवतो केसधातुं पञ्चपतिट्ठितेन वन्दित्वा अत्तनो कायारूळ्हपारुपनं ओमुञ्चित्वा दुल्लभं वत दस्सनन्ति वत्वा धातुपूजं अकासि।

# 9. सब्बेपि देवा अत्तनो अनुरूपं पूजं करिंसु। ततो पज्जुन्नो महाराजा नानारतनमयाहि इट्ठकाहि चेतियं बन्धापेसि। चेतियङ्गणम्पि विसोधापेत्वा तस्स समन्ततो पाकारं परिक्खिपित्वा तस्साविदूरे एकं पोक्खरणिं मापेत्वा चतूसु दिसासु निधियो निदहित्वा धातुपूजं अकासि।
# 10. सब्बे खीणासवा ‘इदं चेतियं अनागते सतब्यामं भविस्सति धम्मराजापि इधेव भविस्सती’ति ब्याकरिंसु।
# 11. तं पन चेतियं समन्ततो किळिविथिकादीना थूपस्स विचारितत्ता पच्छा सोण्डायचेतियन्ति पाकटं।

**पज्जुन्नथूपवंसो निट्ठितो।**
