**वरुणनागराजथूपवंसो**

# 1. ततो पाचिनदिसाभगेन गङ्गातीरे एकं रमणीयं ठानं अत्थि। सब्बे खीणासवा तत्थ गन्त्वा इदं खो आवुसो ठानं छायुदकसम्पन्नं, ‘इमस्मिं ठाने एकं धातुं ठपेतुं वट्टती’ति।
# 2. अथ खो ञाणपण्डितथेरो ‘दायकविचिनने मम भारो’ति वत्वा एकं जाणुमण्डलं पथवियं निधाय अञ्जलिं सिरसि पतिट्ठापेत्वा गाथमाह -

“यो धीरो लोकनायको असमो पुरिसासभो।

चे अपि धातु तस्सेव लभेय्यं वत दायक”न्ति॥

# 3. एवं निपच्चकारं कत्वा कथेन्तस्सेव एको नागराजा नामेन वरुणो नाम सो नागभवनतो सपरिवारेन आगन्त्वा थेरस्स पुरतो पातुरहोसि।
# 4. तदा नागराजा थेरे वन्दित्वा आगमनकारणं पुच्छित्वा ‘भगवतो धातुया ठपनत्थाय एव इधागतम्हा उपासका’ति वुत्ते ‘तेनहि भन्ते धातुचेतिये मम भारं करोथा’ति याचित्वा तेहि दिन्नोकासो हुत्वा सकपरिसं आमन्तेत्वा पहट्ठमानसो हुत्वा धातुगब्भत्थाय पथविं सोधापेत्वा आवाटं खणापेत्वा धातुगब्भं देवविमानसदिसं कारापेत्वा तस्स मज्झे एकरतनमयं पल्लङ्कं मापेत्वा सेतच्छत्तं उस्सापेत्वा धातुचङ्कोटकं सिरसा सम्पटिच्छित्वा एकं अनग्घं मणिकरण्डकं कत्वा परमुक्कं सपत्तं आदरं कत्वा थेरानं रूचिया अनुरूपमेव ठपेसि।
# 5. तं खणञ्ञेव हेट्ठा वुत्तप्पकारानिपि सब्बपाटिहारियानि पातुरहेसुं।
# 6. नागराजापि उदग्गचित्तो हुत्वा अत्तनो परिभोगं अपरम्पि एकं मणिरतनं चक्कवत्तिरञ्ञो मणिसदिसं अनग्घं सब्बकामददं अत्तनो गीवतो ओमुञ्चित्वा भगवतो धातुपूजं अकासि।
# 7. सब्बे नागपरिसा नच्चन्ता गायन्ता सब्बतूरियाभि वादेन्ता उग्घोसिंसु। सब्बे देवा नानप्पकारं साधुकारं अदंसु।
# 8. तस्मिं खणे नागो धातुथूपुं परमदस्सनीयं कत्वा बन्धापेसि।
# 9. निट्ठिते थूपे नागो पदक्खिणं कत्वा ओनतसीसो हुत्वा वन्दित्वा इमं गाथमाह -
    “कप्पकोटिहि यं बुद्धं, दुल्लभं अमतेपि तं।

धातुया दायको अहं, भवामि कुसलत्थिको”ति॥

# 10. एवं वत्वान वरुणो नागराजा महिद्धको पहट्ठचित्तो पक्कमि नागेहि परिवारितोती।

   वरुणनागराजथूपवंसो निट्ठितो।
